A1.Gardener and the Capybaras (easy version)

easy version 长度很小
枚举a的终点，即知道了b的起点，再枚举b的终点，即可知道c的起点，而c的终点就是字符串的最后一位

代码：

bool check(string a,string b,string c){
    return ((b >= a && b >= c) || (b <= a && b <= c));
}

void Sol(){
    string str;cin >> str;
    bool sign = true;
    int len = str.length();
    string a,b,c;
    for(int i = 1;i <= len-2;i++){
        a = str.substr(0, i);
        for(int j = 1;j <= len-1-i;j++){
            b = str.substr(i, j),c = str.substr(i+j, len-(i+j));
            if(check(a,b,c)){
                sign = true;
                break;
            }
        }
        if(sign) break;
    }
    if(sign)
        cout << a << " " << b << " " << c << "\n";
    else 
        cout << ":(\n";
}

A2.Gardener and the Capybaras (hard version)

hard version n <= 200000
继续 O(n^2) 枚举肯定是会超时了，通过题意可以发现如下规律：

  1. 字符串a,b,c中，b中含’a’时，可以假设 b为最小 的，b中如果 只有一个’a’ 的话，一定满足 b<=a && b<=c
  2. b中含’b’时，可以假设 b为最大 的，可以 令str[1…n-2]为b，s[1]肯定为’b’，如果有’a’，那么考虑规律1，此时，a = s[0],c = s[n-1]，b一定满足 b>=a && b >=c

代码：

void Sol(){
    string str;cin >> str;
    bool sign = 0;
    int len = str.length();
    string a, b, c;
    for(int i = 1;i <= len-2;i++){
        if(str[i] == 'a'){
            sign = true;
            a = str.substr(0, i),b = "a",c = str.substr(i+1, len-i);
            break;
        }
    }
    if(!sign){
        int scr = 0;
        for(int i = 1;i <= len-2;i++){
            if(str[i] == 'b'){
                scr = i;
                break;
            }
        }
        if(scr){
            sign = true;
            a = str.substr(0, scr),b = str.substr(scr, len-1-scr),c = str.substr(len-1, 1);
        }
    }
    if(sign)
        cout << a << " " << b << " "<< c << "\n";
    else
        cout << ":(\n";
}

Stack & Queue

1.栈

代码：

const int N = 1e6 + 10;
int stk[N],tt = 0; // tt 栈顶

void insert(int x){
    stk[++tt] = x;
}

void pop(){
    tt--;
}

int head(){
    return stk[tt];
}

bool empty(){
    return tt > 0; // true 则不空
}

2.队列

(1) 普通队列

代码：

const int N = 1e6 + 10;
int q[N],head = 0,tail = 1;

void insert(int x){
    q[++tail] = x;
}

void pop(){
    head++;
}

int front(){
    return q[head];
}

bool empty(){
    return head <= tail; // true 则不空
}

(2) 循环队列

代码：

const int N = 1e6 + 10;
int q[N],head = 0,tail = 0;

void insert(int x,int n){
    q[tail++] = x;
    if(tail == n) tail = 0; // 队尾是所有待输入元素的最后一个则让其和队首相接
}

void pop(int n){
    head++;
    if(head == n) head = 0;
}

int front(){
    return q[head];
}

bool empty(){
    return head != tail; // true 则为空
}

Disjoint Set Union

并查集 是一种 树型 的数据结构，用于处理一些 不相交集 的合并及查询问题

主要有如下两种操作：

  1. 合并两个集合（merge）
  2. 判断两个元素是否在同一个集合中（find）

代码：

namespace DSU{
    const int N = 1e6 + 10;
    int fa[N],r[N];
    // 初始化
    void init(int n){
        for(int i = 1;i <= n;i++) 
            fa[i] = i,r[i] = 1;
    }
    // 查找（路径压缩）
    int find(int x){
        if(x == fa[x]) return x;
        else {
            fa[x] = find(fa[x]);
            return fa[x];
        }
    }
    // 合并（按秩合并）
    void merge(int x,int y){
        int fx = find(x),fy = find(y);
        if(r[fx] <= r[fy]) fa[fx] = y;
        else fa[fy] = fx;
        if(r[fx] == r[fy] && fx != fy) r[y]++;
    }
}

用法：

DSU::init(n);
DSU::find(x);
DSU::merge(x,y);
